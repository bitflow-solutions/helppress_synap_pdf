0515 실시간 조건검색

 

▣ 화면정의

0500 화면에서 저장한 사용자 조건을 실시간으로 감시하여 조건을 만족하는 종목에 대한 진입/탈퇴/재진입을 확인할 수 있는 화면 입니다. 검색된 종목에 대해서 진입 시점을 기준으로 한 수익률 데이터를 확인 할 수 있으며 특정그룹/보유종목/특정종목을 지정하여 감시 할 수 있습니다.
